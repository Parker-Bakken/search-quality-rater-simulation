# Search Intent Types

Understanding user intent is essential for evaluating search results.

---

## Informational Intent
User wants to learn about a topic.

Examples:
how to reset Windows password  
what causes thunderstorms  

---

## Navigational Intent
User wants to reach a specific website or page.

Examples:
youtube login  
amazon homepage  

---

## Transactional Intent
User wants to purchase or acquire something.

Examples:
buy iphone 13  
cheap airline tickets  

---

## Local Intent
User wants results related to a geographic area.

Examples:
coffee shops near me  
spokane public library hours  

---

## YMYL Queries
Queries affecting health, finances, or safety.

Examples:
symptoms of shingles  
best credit cards
